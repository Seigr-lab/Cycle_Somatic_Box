# Step 20: Connect to EcoFlow

**What you're doing**: Connect your power hub to an EcoFlow Power Kit for practical electricity storage and usage.

## EcoFlow Integration

**Why EcoFlow Power Kit**:

- Handles variable renewable energy inputs
- Built-in battery storage
- Standard AC and DC outputs
- Mobile app monitoring

**Input compatibility**:

- Accepts 11-48V DC input
- Your power hub should output in this range
- MPPT charging optimizes power harvest

## Connection Setup

**DC input connection**:

- Use Power Kit's DC input terminals
- Match polarity carefully (positive to positive)
- Use appropriate wire gauge for current

**Input protection**:

- Install DC disconnect switch
- Add fuse protection at Power Kit
- Label connections clearly

## Configuration

**Power Kit settings**:

- Set input voltage range to match your system
- Configure charging parameters
- Enable MPPT mode for maximum efficiency

**Monitoring setup**:

- Connect to EcoFlow mobile app
- Monitor input power and battery charging
- Track daily energy harvest

## System Integration

**Backup power**:

- Power Kit provides backup when generation is low
- Battery storage for nighttime use
- Consistent power output regardless of weather

**Load management**:

- Connect your household loads to Power Kit
- AC outlets for standard appliances
- DC outlets for efficient DC loads

## Testing Integration

**Initial connection test**:

- Start with no loads connected
- Verify Power Kit recognizes input
- Check charging indicators

**Load testing**:

- Add small loads gradually
- Monitor power flow and battery status
- Verify system handles varying generation

- [ ] EcoFlow Power Kit system

- [ ] Input fuses and protection
- [ ] Mobile device for monitoring app
